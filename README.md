# Опис проєкту

Даний проєкт є інтерактивним навчальним вебресурсом, присвяченим вивченню теми **«NoSQL бази даних: введення»** для учнів 10–11 класів.

Ресурс створено з використанням сучасних інструментів генеративного штучного інтелекту та опубліковано за допомогою GitHub Pages.

Метою проєкту є формування в учнів базових знань про сучасні технології зберігання даних, ознайомлення з принципами роботи NoSQL-баз даних та розвиток навичок аналізу інформаційних систем.

---

# Актуальність теми

У сучасному світі щодня створюються величезні обсяги інформації.

Соціальні мережі, онлайн-магазини, стримінгові сервіси, мобільні додатки та хмарні платформи використовують технології зберігання даних нового покоління.

Традиційні реляційні бази даних не завжди можуть ефективно працювати з такими обсягами та різноманітністю інформації.

Саме тому виникла потреба у використанні NoSQL-баз даних, які забезпечують:

- високу швидкодію;
- масштабованість;
- гнучку структуру даних;
- підтримку великих даних (Big Data).

---

# Мета проєкту

Створити інтерактивний освітній ресурс для вивчення основ NoSQL-баз даних учнями старшої школи.

---

# Завдання проєкту

- пояснити поняття NoSQL;
- ознайомити учнів з типами NoSQL-баз даних;
- продемонструвати практичні приклади використання;
- сформувати навички роботи з JSON-документами;
- створити систему самоконтролю знань;
- забезпечити доступ до додаткових ресурсів для самостійного навчання.

---

# Цільова аудиторія

Проєкт орієнтований на:

- учнів 10–11 класів;
- усіх, хто починає вивчати сучасні бази даних.

---

# Очікувані результати навчання

Після опрацювання матеріалу учень повинен:

## Знати:

- що таке NoSQL;
- відмінності між SQL та NoSQL;
- основні типи NoSQL-баз даних;
- сфери застосування NoSQL.

## Вміти:

- аналізувати структуру даних;
- обирати відповідний тип бази даних;
- створювати прості JSON-документи;
- виконувати базові операції з документами.
